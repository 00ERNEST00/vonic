## Getting started

quick start with vue-cli.
```
$ npm i vue-cli -g
$ vue init wangdahoo/vonic-template#2.0 my-vonic-app
$ cd my-vonic-app
$ yarn install
```

Run project
```bash
$ npm run dev
```

That's it.
